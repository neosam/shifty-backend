# Phase 8.3 Plan 01: Foundation — Halbtag-Datenstrukturen, DTO-Surface, i18n-Slots

**Wave-0-Foundation für Halbtag-Support: DB-Migration (additiv, `NOT NULL DEFAULT 'full'` mit `CHECK`), `DayFraction[Entity|TO]`-Enums entlang aller drei Layer (DAO / Domain / DTO), `day_fraction`-Feld auf den drei `AbsencePeriod`-Pendants, OpenAPI-Surface-Pin, Frontend-State-Mirror und 13 i18n-Key-Enum-Slots ohne Bodies. Plan 02..06 hängen alle an dieser Foundation.**
