   ```rust
   //! WebDAV client wrapper (Phase 48 EXP-01 + EXP-03 retry).
   //!
   //! Pure client: takes bytes + folder + filename, uploads to Nextcloud via
   //! WebDAV Basic Auth. In-run retry with exponential backoff (2s/4s/8s per
   //! Q7 in CONTEXT). No DAO, no config-service dep — the scheduler in 48-04
   //! owns the config and wires it in.
   ```
   - `#[derive(Debug, thiserror::Error)] pub enum WebDavError { #[error("transient webdav error after {attempts} attempts: {reason}")] Transient { attempts: usize, reason: Arc<str> }, #[error("permanent webdav error: {status} {body}")] Permanent { status: u16, body: Arc<str> }, #[error("http/io error: {0}")] Io(#[from] reqwest::Error) }`
   - `pub struct WebDavClient { client: reqwest::Client, base_url: Arc<str>, retry_delays: Arc<[std::time::Duration]> }`
   - `pub fn new(base_url: impl Into<Arc<str>>, user: &str, app_token: &str) -> Result<Self, WebDavError>` — baut `reqwest::Client` mit Basic-Auth-Middleware oder setzt `Authorization: Basic base64(user:token)` als Default-Header via `HeaderMap`; setzt sinnvollen Timeout (`.timeout(Duration::from_secs(30))`) und `rustls`-TLS-Backend.
   - `pub fn new_with_delays(base_url, user, app_token, delays: impl Into<Arc<[Duration]>>) -> Result<Self, WebDavError>` — test-only-freundliche Variante mit injectable delays.
   - `pub async fn upload_file(&self, folder: &str, filename: &str, bytes: Vec<u8>) -> Result<(), WebDavError>`:
     - Erst `mkcol_if_missing(folder).await?` (MKCOL, 405 wird geschluckt).
     - Dann PUT `${base_url}/${folder}/${filename}` mit body = bytes.
     - Retry-Loop um beide Requests: `for delay in self.retry_delays.iter() { match do_one_pass().await { Ok(()) => return Ok(()), Err(Transient) => sleep(delay).await, Err(Permanent) => return Err(...) } }` — nach letztem Fehl-Versuch `return Err(WebDavError::Transient { attempts: N, reason })`.
     - Der Retry umfasst **eine ganze upload_file-Passage** (MKCOL + PUT), nicht MKCOL und PUT einzeln — ein transienter Fehler auf MKCOL retried die gesamte Sequenz.
   - `fn classify(status: reqwest::StatusCode, on_mkcol: bool) -> Classification` — private pure fn, testbar.
   - `pub mod webdav_client;` in `service_impl/src/lib.rs` (pub — Scheduler in 48-04 nutzt).
