**Schritt 1: Cargo.toml-Swap.**

In `/home/neosam/programming/rust/projects/shifty/shifty-dioxus/Cargo.toml` die Zeilen 28-29 ersetzen.

VORHER (exakte Zeilen):
```toml
[dependencies.rest-types]
path = "rest-types"
```

NACHHER (exakter Ersatz):
```toml
[dependencies.rest-types]
path = "../rest-types"
default-features = false
```

Begründung `default-features = false`: Backend-`rest-types/Cargo.toml` hat `default = []`, also ist es technisch ein No-Op. ABER: Falls in einer zukünftigen Backend-Phase ein `default = ["service-impl"]`-Default eingeführt wird, schützt diese explizite Deklaration den Frontend-WASM-Build vor dem unbeabsichtigten Pull-In des `service`-Crates. Es ist Zukunftssicherheit (Defense-in-Depth).
