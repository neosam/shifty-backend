**Schritt C — Pro-Woche-Closure:**
```rust
let absence_derived_balance_hours = if expected_hours <= 0.0 {
    0.0f32
} else {
    derived.iter()
        .filter(|(d, _)| { let w = ShiftyDate::from(**d).as_shifty_week(); w.year == year && w.week == week })
        .map(|(_, r)| r.hours)
        .sum::<f32>()
};
```

Setzt `absence_derived_balance_hours: 0.0` im dynamic-Zweig, den berechneten Wert im else-Zweig.
