pass today. This plan VERIFIES the fix is present and ADDS the missing
argument-pinning regression test so the bug cannot silently reappear.

    .await?;

    .filter(|c| c.deleted.is_none())
    .map(|c| c.vacation)
    .unwrap_or(0);
