**1. shifty-dioxus/src/api.rs — get_voluntary_stats (Zeilen 405-425):**

Signatur ergaenzen: `until_week: u8` zusaetzlich. URL-Format:

```rust
/// Phase 54 Gap-Closure G1 (VOL-STAT-01/02): HR-only voluntary-hours stats
/// mit echter Date-Range. Range wird aus dem Employee-Report-Kontext abgeleitet:
/// from = Jan 1 (year), to = Sonntag der (year, until_week). Bei until_week > 
/// weeks_in_year clampt der Aufrufer (Task 2).
pub async fn get_voluntary_stats(
    config: Config,
    sales_person_id: Uuid,
    year: u32,
    until_week: u8,
) -> Result<VoluntaryStatsTO, reqwest::Error> {
    info!("Fetching voluntary stats");
    // from_date = 1. Januar des Jahres (ISO YYYY-MM-DD).
    let from_date = format!("{:04}-01-01", year);
    // to_date = Sonntag der (year, until_week). Falls die ISO-Woche im 
    // Folge-Kalenderjahr endet (Standard), fangen wir mit time::Date-Rechnung
    // ab. Fallback bei ungueltiger Woche: 12-31 des Jahres.
    let to_date = compute_to_date(year, until_week);
    let url = format!(
        "{}/report/{}/voluntary-stats?from_date={}&to_date={}",
        config.backend, sales_person_id, from_date, to_date
    );
    let response = reqwest::get(url).await?;
    response.error_for_status_ref()?;
    let res = response.json().await?;
    info!("Fetched voluntary stats");
    Ok(res)
}

/// Berechnet das to_date (Sonntag der ISO-Woche `until_week` im Jahr `year`)
/// im Format YYYY-MM-DD. Fallback bei ungueltiger Woche: "YYYY-12-31".
fn compute_to_date(year: u32, until_week: u8) -> String {
    if let Ok(sunday) = time::Date::from_iso_week_date(
        year as i32,
        until_week,
        time::Weekday::Sunday,
    ) {
        format!("{:04}-{:02}-{:02}", sunday.year(), sunday.month() as u8, sunday.day())
    } else {
        format!("{:04}-12-31", year)
    }
}
```
