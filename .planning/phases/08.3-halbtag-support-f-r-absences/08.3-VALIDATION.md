> Per-phase validation contract for feedback sampling during execution.

## Test Infrastructure

| Property | Value |
|----------|-------|
| **Framework** | Rust `cargo test` (built-in) + SQLite in-memory (sqlx) + mockall + dioxus-ssr (Frontend-Snapshot) |
| **Config file** | `Cargo.toml` (workspace root) + `shifty-dioxus/Cargo.toml` |
| **Quick run command** | `cargo test --workspace --lib` |
| **Full suite command** | `cargo check --workspace && cargo test --workspace && (cd shifty-dioxus && cargo build --target wasm32-unknown-unknown)` |
| **Estimated runtime** | Quick ~20-40 s; Full ~90-180 s (WASM-Build kostet am meisten) |

## Sampling Rate

- **After every task commit:** Run `cargo test --workspace --lib` (Quick) — feedback latency < 60 s
- **After every plan wave:** Run Full suite (`cargo check --workspace && cargo test --workspace && WASM-build`)
- **Before `/gsd-verify-work`:** Full suite green inkl. WASM-Build und Snapshot-Locking-Test
- **Max feedback latency:** 60 s pro Task / 180 s pro Wave
